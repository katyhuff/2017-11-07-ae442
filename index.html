An academic reactor or reactor plant almost always has the following basic characteristics: (1) It is simple. (2) It is small. (3) It is cheap. (4) It is light. (5) It can be built very quickly. (6) It is very flexible in purpose. (7) Very little development will be required. It will use off-the-shelf components. (8) The reactor is in the study phase. It is not being built now.

On the other hand a practical reactor can be distinguished by the following characteristics: (1) It is being built now. (2) It is behind schedule. (3) It requires an immense amount of development on apparently trivial items. (4) It is very expensive. (5) It takes a long time to build because of its engineering development problems. (6) It is large. (7) It is heavy. (8) It is complicated.

How Much Fuel?

First, let's think about the mass of fuel question. I could state without proof that thermal \(^{235}U\) fissions consume about 1.23g per MWd. But, let's prove it.

First, what is fission :

\[ \begin{align} ^{235}_{92}U + ^{1}_{0}n \rightarrow \left(^{236}_{92}U\right)^* \rightarrow ^{A_1}_{Z_1}X + ^{A_2}_{Z_2}X + {\nu ^{1}_{0}n} + Q \end{align} \]

That's our fission reaction. For each fission, 200MeV of energy is released.

Let's then calculate how many atoms result in a thermal power of one Joule (1 \(W\cdot s\)).

\[ \begin{align} \left(\frac{1\mbox{ fission}}{200 MeV}\right) \left(\frac{1MeV}{1.6021\times10^{-13}J}\right) & =3.12\times 10^{10} \left(\frac{fissions}{J}\right) \end{align} \]

Each fission is one uranium atom, so:

\[\begin{align} \left(\frac{3.12\times 10^{10}\mbox{atoms fissioned}}{J}\right) \left(235 \frac{g}{ mol}\right)\left(\frac{1 mol}{6.022\times10^{23} atoms}\right)& = 1.22\times 10^{-11}\frac{g}{J}\\ \end{align} \]

We need to convert to MWd, so we'll need to know how many joules are in a MWd.

\[ \begin{align} \left(\frac{1 J}{W\cdot s}\right) \left(\frac{10^6 W}{MW}\right) \left(\frac{60\times60 s}{h}\right) \left(\frac{24h}{d}\right) &= 8.64\times10^{10}\frac{J}{MWd} \end{align} \]
Then, we can multiply J/MWd against g/J to get g/MWd. \[ \begin{align} \left(8.64\times10^{10}\frac{J}{MWd}\right)\left(1.22\times 10^{-11}\frac{g}{J}\right)&= 1.05 \frac{g}{MWd} \end{align} \]

However, we have to correct for the fact that fuel is also consumed by non-fission capture.

\[ \begin{align} \frac{\sigma_c}{\sigma_f} &= 0.169\\ \Rightarrow\frac{\sigma_f + \sigma_c}{\sigma_f} &= 1.169\\ \end{align} \]

So, we get:

\[ \begin{align} \left(\frac{g}{MWd}\right)\left(\frac{\sigma_f + \sigma_c}{\sigma_f}\right) &= 1.05 \frac{g}{MWd}\times 1.169 = 1.23 \frac{g}{MWd} \end{align} \]
  • If your fuel is 90% enriched, that means it's 90% \(^{235}U\) and approximately 10% \(^{238}U\).
  • Don't forget to include the other elements in the fuel (if it's \(UO_2\), some of fuel mass comes from oxygen).
  • You're probably going to need some coolant. liquid metals are heavier than water, but allow a fast spectrum, so you can breed and fission a tiny bit of \(^{239}Pu\).
  • Finally, you're going to need reflectors (Be) to keep the neutrons in the reactor.
